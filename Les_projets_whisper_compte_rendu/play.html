Projet réalisé sous Play! framework.

Play est un framework open source. Il permet de réaliser des applications web en Java et/ou en Scala. Dans le cadre du projet, Play2 a été utilisé pour coder le site en Java, avec le moteur de template Scala.

Le Back-end

Play!

Contrairement aux applications JEE, les applications Play ne sont pas contruites en "couches", ce qui rend le codage plus rapide et plus intuitif de premier abord. Ce framework permet de réaliser des applications RESTfull car il met en place une architecture REST. Ce framework MVC permet une architecture modulaire. Play2 combine à la fois du Java et du Scala. Le serveur interne à Play est JBoss Netty. Comme Play est un framework REST, il est possible de réaliser des opérations CRUD. L'application renvoie du XML et du JSON.


Le Front-end

Twitter Bootstrap

En ce qui concerne le design du site, nous avons utilisé le kit CSS Twitter Bootstrap. C'est un framework de front-end qui repose sur un système de grilles et contient des templates déja prêtes à l'emploi, permetant de réaliser des sites web très esthétiques.
